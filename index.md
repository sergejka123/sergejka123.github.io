## ‚öôÔ∏è How to Use Syncord
1. **Upload a File**
   - To upload a file, use the command:
     ```
     syncord upload path/to/your/file.txt
     ```
   - Syncord will confirm the upload status.

2. **Download a File**
   - To download a file you have previously uploaded, use:
     ```
     syncord download filename
     ```
   - This will retrieve the file from your Syncord backup.

3. **List Files**
   - To see all the files you have stored, run:
     ```
     syncord list
     ```
   - This command displays the names of your files stored in Discord.
